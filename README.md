# printticket module

##目录简介

* src/main
    + src/main/java/code 主要类
* src/main
    + src/main/resources 存放数据信息的json文件
        - src/main/resources/good_info.json 存储商店所有商品信息
        - src/main/resources/discount_info.json 存储目前支持的折扣信息
    + src/main/java   主要逻辑源代码
        - /code  主要的类
        - /DiscountTypes  打折信息类
        - /util  工具类


##用到的设计模式

+ 工厂模式
+ 单例模式
